# Commentaires

Partagez vos impressions, signalez des bugs ou demandez des fonctionnalites directement depuis MyCompanyDesk.

## Soumettre un commentaire

1. Cliquez sur votre **menu de profil** dans la barre laterale (ou le menu mobile)
2. Selectionnez **Feedback**
3. Choisissez une **categorie** :
   - **General** -- Commentaires ou questions generales
   - **Bug** -- Signalez quelque chose qui ne fonctionne pas correctement
   - **Fonctionnalite** -- Suggerez une nouvelle fonctionnalite ou amelioration
   - **Ergonomie** -- Partagez vos impressions sur l'experience utilisateur
4. Evaluez optionnellement votre experience (1-5 etoiles)
5. Redigez votre **message** decrivant le commentaire
6. Cliquez sur **Envoyer**

Apres l'envoi, vous verrez un ecran de confirmation. Votre commentaire est transmis a l'equipe MyCompanyDesk pour examen.

## Ce qui se passe ensuite

L'equipe examine toutes les soumissions :

- **Les signalements de bugs** sont priorises et investigues
- **Les demandes de fonctionnalites** sont prises en compte pour la feuille de route produit
- **Les retours d'ergonomie** aident a ameliorer l'interface et les flux de travail

## Conseils

- Soyez aussi specifique que possible -- decrivez ce que vous faisiez et ce que vous attendiez
- Pour les bugs, mentionnez la page ou le probleme s'est produit (ceci est capture automatiquement)
- Vous pouvez soumettre des commentaires depuis n'importe quelle page de l'application
