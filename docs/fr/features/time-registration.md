# Agenda

Enregistrez vos heures de travail, planifiez vos journées et transformez le temps facturable en factures. La page **Agenda** dans la barre latérale combine l'enregistrement du temps avec un agenda : vous voyez le plan du jour, les entrées planifiées à côté des heures enregistrées et des suggestions issues de vos calendriers connectés, le tout au même endroit.

## La page en un coup d'œil

Passez d'une vue à l'autre avec le sélecteur en haut (balayez entre les périodes sur mobile) :

- **Jour** : le plan et les heures enregistrées du jour. Selon vos réglages, vous voyez une chronologie ou une liste compacte, avec les entrées planifiées séparées des entrées enregistrées. Les événements de vos calendriers connectés apparaissent à côté de vos entrées et se convertissent en entrées de temps d'un simple appui, et vous recevez des suggestions basées sur votre activité récente.
- **Semaine** : sur ordinateur, un planning sur sept jours où les blocs pleins sont des heures enregistrées et les blocs hachurés des heures planifiées ; cliquez sur un créneau vide pour ajouter une entrée. Sur mobile, un résumé par jour à explorer d'un appui.
- **Mois** : les totaux par jour ; sélectionnez un jour pour y accéder.
- **Liste** : un tableau consultable de toutes les entrées avec des filtres par statut de facturation, client, projet et déplacements, plus les totaux de la sélection en cours.

Les entrées planifiées portent le badge **Provisoire** ; appuyez sur **Confirmer** une fois le travail réellement effectué pour les transformer en heures enregistrées.

Vous préférez ne pas le faire manuellement ? Activez **Confirmer automatiquement le temps provisoire** dans les paramètres de l'agenda. Les entrées provisoires sont alors confirmées automatiquement une fois leur date prévue passée, même si personne n'ouvre l'agenda.

## Enregistrer du temps

### Minuteur

Sur mobile, la vue Jour comprend un minuteur : lancez-le quand vous commencez à travailler et arrêtez-le pour enregistrer le temps écoulé comme entrée. Vous pouvez faire du minuteur votre mode de travail par défaut via le réglage du mode de travail (voir Paramètres ci-dessous).

### Entrées manuelles

1. Cliquez sur **Ajouter une entrée** (raccourci clavier A, ou le bouton + sur mobile)
2. Le panneau de saisie rapide s'ouvre : choisissez le client et éventuellement un projet, saisissez vos heures et ajustez la description et le tarif si nécessaire
3. Saisissez le temps comme un nombre total d'heures, ou passez à une heure de début et de fin
4. Enregistrez l'entrée

### Description de ligne par défaut

Quand vous ajoutez une entrée de temps, le champ de description est automatiquement pré-rempli à partir de vos descriptions de ligne par défaut. Le système vérifie dans cet ordre :

1. La description de ligne par défaut du projet
2. La description de ligne par défaut du client
3. La valeur par défaut de l'espace de travail

Votre propre saisie n'est jamais écrasée. Une fois que vous avez tapé une description personnalisée, la valeur pré-remplie ne la remplace plus.

### Mode heures uniquement

Vous préférez enregistrer seulement un total par jour ? Activez le **mode Heures uniquement** dans les paramètres de l'agenda. Il masque la chronologie et les champs d'heure de début et de fin, pour que vous ne saisissiez que le total d'heures par jour. Le tarif et le champ facturable restent disponibles.

## Facturer vos heures

### Créer une facture depuis la page Agenda

Quand vous avez des entrées non facturées, cliquez sur **Créer une facture**. Un panneau s'ouvre pour choisir un client ; il liste toutes les entrées non facturées de ce client avec leur total. Confirmez, et une facture brouillon est créée avec une ligne par entrée. Le temps de déplacement et les frais de déplacement facturables liés à ces entrées sont ajoutés comme lignes séparées.

### Choisir des entrées individuelles sur le formulaire de facture

Vous voulez facturer seulement certaines entrées ? Créez ou modifiez directement une facture : le formulaire de facture dispose d'une section temps qui liste les entrées non facturées du client sélectionné, pour que vous choisissiez exactement celles à inclure.

### Descriptions de ligne sur la facture

Les lignes de facture sont automatiquement décrites : la description de l'entrée elle-même est utilisée en premier, sinon le nom du projet, sinon la période. Un modèle de description par client (défini sur la page client) remplace ce format.

### Facturation automatique du temps

La facturation automatique se configure par client. Sur la page du client, activez **Facturation automatique** pour créer automatiquement chaque mois une facture pour le temps enregistré de ce client, envoyée automatiquement si vous le souhaitez. La facture est créée comme brouillon et reste ouverte pour que vous puissiez la relire avant l'envoi.

## Actions groupées

Sélectionnez plusieurs entrées dans la vue liste (appui long sur mobile) pour agir dessus en une fois :

- **Marquer comme facturable** ou **Marquer comme non facturable**
- **Archiver**
- Supprimer

## Synchronisation avec calendrier externe

Connectez Google Agenda ou Outlook Calendar pour réunir votre agenda et vos heures. Ouvrez les paramètres de l'agenda via l'engrenage et suivez le lien calendrier, ou allez directement à la page des calendriers connectés. Vous pouvez y :

- Connecter **Google Agenda** ou **Outlook Calendar**
- Activer la synchronisation par connexion avec **Activer la synchronisation**
- Choisir le **Sens de synchronisation** : **Vers le calendrier** (vos heures enregistrées apparaissent dans votre calendrier), **Depuis le calendrier** (vos rendez-vous apparaissent sur la page Agenda, prêts à être enregistrés) ou **Les deux**
- Activer un flux **Abonnement calendrier (iCal)** en lecture seule pour suivre vos heures enregistrées depuis n'importe quelle appli de calendrier

Les rendez-vous provenant d'un calendrier connecté s'affichent dans les vues Jour et Semaine ; appuyez dessus pour en faire une entrée de temps.

## Paramètres

La roue dentée sur la page Agenda ouvre les paramètres de l'agenda. Vous y configurez :

- Le **mode Heures uniquement**, la **Confirmation automatique du temps provisoire**, et le reste du comportement de saisie
- Le mode de travail : **Heures**, **Quarts** ou **Minuteur**
- Les heures de travail affichées sur la chronologie
- Les étapes affichées par la saisie rapide (projet, notes, déplacements)
- Le tarif horaire par défaut (administrateurs d'équipe)
- Les valeurs par défaut des déplacements comme les adresses, le véhicule et le tarif kilométrique
- Utiliser votre localisation actuelle comme point de départ d'un déplacement, quand votre appareil le permet
- Un lien pour connecter un calendrier externe

## Conseils

- Enregistrez vos heures quotidiennement pour des relevés précis ; les suggestions permettent de re-saisir un travail récurrent en un appui
- Planifiez votre semaine à l'avance avec des entrées provisoires et confirmez-les au fur et à mesure
- Vérifiez régulièrement votre temps non facturé, pour ne laisser passer aucune heure facturable
- Connectez votre calendrier une fois, et vos rendez-vous deviennent automatiquement des entrées enregistrables
