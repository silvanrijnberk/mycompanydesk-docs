# Facturation et abonnements

Gerez votre abonnement MyCompanyDesk, consultez votre utilisation et mettez a niveau votre plan.

## Abonnements

MyCompanyDesk propose trois plans :

### Gratuit

Commencez sans frais :

- Factures et clients limites
- Suivi des depenses basique
- Rapports basiques
- 1 utilisateur
- Support communautaire

### Pro

Pour les entreprises en croissance :

- Factures et clients illimites
- Suivi complet des depenses avec numerisation de recus
- Suggestions IA
- Image de marque PDF personnalisee
- Export CSV
- Jusqu'a 5 membres d'equipe
- Support par e-mail

### Business

Pour les equipes et les operations en expansion :

- Tout ce qui est inclus dans Pro
- Membres d'equipe illimites
- Acces API
- Support prioritaire
- Integrations avancees

Les plans sont disponibles en intervalles de facturation **mensuelle** et **annuelle**. Les plans annuels incluent une remise.

## Gerer votre abonnement

### Mettre a niveau

1. Allez dans **Parametres > Facturation**
2. Cliquez sur **Mettre a niveau** sur le plan souhaite
3. Completez le paiement via Stripe
4. Les fonctionnalites sont activees immediatement

### Reduire le plan

1. Allez dans **Parametres > Facturation**
2. Cliquez sur **Changer de plan**
3. Selectionnez le plan inferieur
4. Les changements prennent effet a la fin de la periode de facturation en cours

### Annuler

1. Allez dans **Parametres > Facturation**
2. Cliquez sur **Gerer l'abonnement** pour ouvrir le portail de facturation Stripe
3. Annulez votre abonnement
4. Vous conservez l'acces jusqu'a la fin de la periode payee

## Suivi de l'utilisation

Consultez votre utilisation actuelle sur la page de facturation :

- **Factures creees** ce mois-ci
- **Clients** dans votre espace de travail
- **Membres de l'equipe** actifs
- **Appels API** (plan Business)

L'utilisation est reinitialise mensuellement. Si vous approchez d'une limite, vous verrez une notification suggerant une mise a niveau.

## Methodes de paiement

Les paiements sont traites de maniere securisee via **Stripe**. Methodes acceptees :

- Carte de credit (Visa, Mastercard, Amex)
- Carte de debit
- Prelevement SEPA (UE)

Gerez votre methode de paiement dans le portail de facturation Stripe.

## Portail de facturation

Cliquez sur **Gerer l'abonnement** pour acceder au portail de facturation Stripe ou vous pouvez :

- Mettre a jour votre methode de paiement
- Consulter l'historique des factures
- Telecharger les recus
- Modifier l'adresse de facturation

## Droits d'acces aux fonctionnalites

Certaines fonctionnalites sont reservees a certains plans :

| Fonctionnalite | Gratuit | Pro | Business |
|---|---|---|---|
| Image de marque personnalisee | - | Oui | Oui |
| Export CSV | - | Oui | Oui |
| Numerisation de recus | - | Oui | Oui |
| Suggestions IA | - | Oui | Oui |
| Verification de texte | - | Oui | Oui |
| Acces API | - | - | Oui |

Tenter d'utiliser une fonctionnalite reservee sur un plan inferieur affiche une invitation a mettre a niveau.
