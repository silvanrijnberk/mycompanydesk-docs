# Parametres de l'entreprise

Configurez les informations de votre entreprise, l'image de marque, les parametres financiers par defaut et les fonctionnalites activees.

## Acceder aux parametres

Naviguez vers **Entreprise** dans la barre laterale pour acceder a tous les parametres de l'entreprise. Les parametres sont organises en sections.

## Informations de l'entreprise

### Coordonnees de l'entreprise

- **Nom de l'entreprise** -- Le nom enregistre de votre entreprise (apparait sur tous les documents)
- **Adresse** -- Rue, code postal, ville, pays
- **E-mail de contact** -- Adresse e-mail professionnelle principale
- **Telephone** -- Numero de telephone professionnel

### Immatriculation

- **Numero de TVA** -- Votre numero d'identification TVA (par ex., NL123456789B01)
- **Chambre de Commerce** -- Numero d.immatriculation commerciale. Utilisez **Mettre a jour depuis KVK** pour recuperer les dernieres donnees du registre du commerce neerlandais. Les champs vides (nom de l.entreprise, adresse, code postal, ville, pays) sont automatiquement remplis. Les valeurs que vous avez deja saisies restent inchangees. Chaque espace de travail dispose de 100 recherches KVK gratuites par jour (les resultats mis en cache ne comptent pas dans la limite). Lorsqu.il reste 20 recherches ou moins, l.indication du champ affiche un compteur du nombre restant. Si la limite quotidienne est atteinte, vous pouvez saisir les donnees manuellement ou reessayer demain.
- **IBAN** -- Compte bancaire pour recevoir les paiements

::: tip
L'autocompletion d'adresse est disponible -- commencez a taper et selectionnez parmi les suggestions alimentees par Google Maps.
:::

## Image de marque

### Logo

Telechargez le logo de votre entreprise pour l'utiliser sur :

- Les PDF de factures et devis
- Les communications par e-mail
- Le portail client
- La page publique d'entreprise

Formats acceptes : PNG, JPG, SVG. Utilisez une image haute resolution avec un fond transparent pour de meilleurs resultats.

### Couleurs

Definissez votre couleur d'accentuation de marque. Elle est utilisee sur :

- Les documents PDF
- Les modeles d'e-mail
- Les pages accessibles aux clients

## Parametres financiers par defaut

### Devise

Definissez votre devise par defaut (EUR par defaut). Elle s'applique aux nouvelles factures et aux nouveaux devis.

### Conditions de paiement

Definissez le nombre de jours par defaut avant l'echeance d'une facture. Valeurs courantes :

- **14 jours** -- Deux semaines
- **30 jours** -- Un mois
- **60 jours** -- Deux mois

Vous pouvez modifier cela pour chaque facture individuellement.

### Numerotation des factures

Personnalisez la generation des numeros de facture :

- **Prefixe** -- Texte avant le numero (par ex., `INV-`, `2024-`)
- **Numero de depart** -- Le premier numero de la sequence
- **Incrementation automatique** -- Les numeros augmentent automatiquement

## Activation des fonctionnalites

Activez ou desactivez les modules optionnels :

| Fonctionnalite | Description |
|---|---|
| **Objets / Actifs** | Gestion de biens locatifs et d'actifs |
| **Contrats** | Contrats de services et de location recurrents |
| **Projets** | Suivi du travail base sur des projets |
| **Devis** | Gestion des devis |
| **Autoliquidation** | Facturation B2B intra-UE |
| **Suggestions IA** | Recommandations de categorisation intelligentes |
| **Numerisation de recus** | Saisie de depenses par OCR |
| **Verification de texte** | Verification de la grammaire et de l'orthographe |

Les modules desactives sont masques dans la navigation pour garder votre espace de travail epure.

## Modele PDF

Personnalisez la mise en page et l'apparence de vos PDF generes :

- Positionnement des champs
- Visibilite des informations
- Police et taille
- Placement du logo

Accedez a l'**editeur PDF** complet pour une personnalisation visuelle par glisser-deposer depuis Entreprise > Editeur PDF.

::: info
La personnalisation avancee du PDF necessite le plan **Pro** ou superieur.
:::
