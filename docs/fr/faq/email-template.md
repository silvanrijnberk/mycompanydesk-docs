Pour personnaliser les modèles d e-mail :
1. Accédez à Paramètres de l espace de travail → Modèles d e-mail
2. Sélectionnez le type de document que vous voulez modifier
3. Modifiez l objet et le corps du message à cet endroit
4. Ajoutez si besoin des variables de modèle
5. Les changements sont enregistrés automatiquement

Envoyez ensuite un document de test afin de vérifier l objet final et le texte du message.
