Pour changer votre langue :
1. Allez dans Mon compte → Langue
2. Sélectionnez votre langue préférée
3. Le changement s'applique immédiatement et recharge la page si nécessaire

Note : La langue des documents se règle séparément via Entreprise → Facturation (langue PDF).
