Les permissions de page définissent ce que chaque membre d'équipe ou comptable peut faire :
• Lecture - consulter les données
• Écriture - modifier les enregistrements existants
• Création - ajouter de nouveaux enregistrements

Pour les configurer :
1. Allez dans Paramètres → « Accès et comptable »
2. Ouvrez « Gérer l'accès » pour le membre ou le comptable concerné
3. Activez ou désactivez Lecture, Écriture et Création pour chaque page
4. Enregistrez les modifications

Remarque : l'accès Lecture est nécessaire avant Écriture ou Création.
