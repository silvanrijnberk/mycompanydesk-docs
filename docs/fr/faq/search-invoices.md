Pour rechercher une facture spécifique :
1. Accédez aux Factures
2. Utilisez la barre de recherche pour taper le numéro de facture, le nom du client ou le montant
3. Utilisez les filtres de statut (Brouillon, Envoyée, Payée, En retard)
4. Cliquez sur la facture pour l'ouvrir

Astuce : Combinez la recherche avec les filtres de statut pour des résultats plus rapides.
