Pour modifier le format de date ou de nombre :
1. Accédez à Mon compte → Préférences
2. Choisissez le format de date ou de nombre à utiliser
3. Les changements sont enregistrés automatiquement

Le nouveau format est appliqué partout dans l application où cette préférence est utilisée.
