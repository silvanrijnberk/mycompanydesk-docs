Pour modifier une facture :
1. Ouvrez la facture depuis la liste
2. Cliquez sur « Modifier » (disponible uniquement pour les brouillons)
3. Modifiez les lignes, client, dates ou conditions
4. Enregistrez la facture mise à jour

Note : Les factures envoyées ne peuvent pas être modifiées directement.
