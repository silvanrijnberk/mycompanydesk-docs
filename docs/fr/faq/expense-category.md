Pour definir ou modifier une categorie de depense :
1. Ouvrez la depense en mode edition
2. Trouvez le menu deroulant « Categorie »
3. Selectionnez la categorie appropriee (ex. Bureau, Voyage, Logiciel)
4. Enregistrez la depense

Utilisez les categories de facon coherente afin que les filtres, listes et rapports restent plus faciles a verifier.
