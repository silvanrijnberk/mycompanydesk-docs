Pour supprimer votre compte :
1. Allez dans Mon compte → Sécurité
2. Exportez d abord vos données via Mon compte → Export & sauvegarde si vous avez besoin d une sauvegarde
3. Dans la zone de danger, cliquez sur « Supprimer »
4. Confirmez la suppression - cette action est définitive

Important : exportez tout ce dont vous avez besoin avant de continuer.
