Pour ajouter ou modifier votre logo :
1. Allez dans Paramètres → Profil entreprise (ou Éditeur PDF)
2. Trouvez la section « Logo »
3. Cliquez sur « Télécharger » et sélectionnez votre fichier logo
4. Enregistrez - votre logo apparaîtra sur tous les documents
