MyCompanyDesk propose des applications natives pour iOS et Android :

- **iPhone et iPad** : téléchargez [MyCompanyDesk sur l'App Store](https://apps.apple.com/nl/app/id6776958981)
- **Android** : téléchargez [MyCompanyDesk sur Google Play](https://play.google.com/store/apps/details?id=com.mycompanydesk.app)

Connectez-vous avec le même compte que sur votre ordinateur ; tout reste synchronisé.

Vous préférez ne pas installer d'application ? L'application web fonctionne aussi très bien dans votre navigateur mobile :
1. Ouvrez votre navigateur mobile (Safari, Chrome, etc.)
2. Accédez à la même URL que sur votre ordinateur
3. L'interface s'adapte automatiquement à la taille de votre écran
4. Vous pouvez l'ajouter à votre écran d'accueil pour un accès rapide

Astuce : toutes les fonctionnalités sont disponibles sur mobile. Créez des factures, enregistrez des dépenses et gérez vos clients en déplacement.
