To edit an invoice:
1. Open the invoice from the list
2. Click "Edit" (only available for draft invoices)
3. Make your changes to line items, customer, dates, or terms
4. Save the updated invoice

Note: Sent invoices cannot be directly edited. You can cancel them and create a new one, or create a credit note for corrections.
