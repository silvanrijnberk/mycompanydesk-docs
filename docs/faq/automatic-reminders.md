To set up automatic payment reminders:
1. Go to Company → Invoicing
2. Enable automatic reminders there
3. Changes in that section save automatically
4. To edit reminder text, go to Workspace Settings → Email Templates and select the reminder template

After that, review a sample overdue invoice and the reminder template so you can confirm how the reminder flow will look in the app.
