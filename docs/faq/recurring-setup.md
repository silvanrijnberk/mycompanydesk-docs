To set up recurring invoices:
1. Go to Recurring Invoices → click "New"
2. Select the customer and add line items
3. Set the frequency and start date
4. Choose whether to auto-send or create as draft
5. Save and review the recurring list to confirm the schedule

For recurring expenses, open Recurring Expenses and follow the same setup flow there.
