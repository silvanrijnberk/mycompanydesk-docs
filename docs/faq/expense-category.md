To set or change an expense category:
1. Open the expense in edit mode
2. Find the "Category" dropdown
3. Select the appropriate category (e.g. Office, Travel, Software, etc.)
4. Save the expense

Use the same category names consistently so filters, lists, and reports stay easier to review.
