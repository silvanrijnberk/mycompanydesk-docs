To customize email templates:
1. Go to Workspace Settings → Email Templates
2. Select the document type you want to edit (invoice, quote, reminder, etc.)
3. Update the subject and body there
4. Optionally insert template variables
5. Changes save automatically

Send yourself a test document after editing so you can verify the final subject and message text.
