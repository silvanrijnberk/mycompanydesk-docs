Um eine bestimmte Ausgabe zu suchen:
1. Gehe zu Ausgaben
2. Verwende die Suchleiste, um Lieferantenname, Betrag oder Beschreibung einzugeben
3. Verwende Filter für Datumsbereich oder Kategorie
4. Klicke auf die Ausgabe, um sie zu öffnen

Tipp: Regelmäßiges Kategorisieren macht Ausgaben viel leichter auffindbar.
