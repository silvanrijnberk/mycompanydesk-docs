So bearbeitest du eine Rechnung:
1. Öffne die Rechnung aus der Liste
2. Klicke auf „Bearbeiten" (nur für Entwürfe)
3. Ändere Positionen, Kunde, Daten oder Bedingungen
4. Speichere die aktualisierte Rechnung

Hinweis: Gesendete Rechnungen können nicht direkt bearbeitet werden.
