MyCompanyDesk bietet native Apps für iOS und Android:

- **iPhone und iPad**: Laden Sie [MyCompanyDesk im App Store](https://apps.apple.com/nl/app/id6776958981) herunter
- **Android**: Laden Sie [MyCompanyDesk bei Google Play](https://play.google.com/store/apps/details?id=com.mycompanydesk.app) herunter

Melden Sie sich mit demselben Konto an wie am Desktop; alles bleibt synchron.

Möchten Sie lieber keine App installieren? Die Web-App funktioniert auch im mobilen Browser einwandfrei:
1. Öffnen Sie Ihren mobilen Browser (Safari, Chrome usw.)
2. Navigieren Sie zur gleichen URL wie am Desktop
3. Die Oberfläche passt sich automatisch an Ihre Bildschirmgröße an
4. Sie können die App für schnellen Zugriff zum Startbildschirm hinzufügen

Tipp: Alle Funktionen sind mobil verfügbar. Erstellen Sie Rechnungen, erfassen Sie Ausgaben und verwalten Sie Kunden von unterwegs.
