So ändern Sie den Stil Ihrer PDF-Rechnungen:
1. Gehen Sie zu Einstellungen → "Rechnungsdesign"
2. Klicken Sie auf "Aus Corporate Design generieren" für ein Design in Ihrer eigenen Markenfarbe, oder wählen Sie eine der Stilvorlagen
3. Passen Sie Akzentfarbe, Papierton, Schriften, Layout und Fußzeile an; die A4-Vorschau aktualisiert sich live
4. Änderungen werden automatisch gespeichert und gelten für alle neuen Rechnungen, Angebote und Erinnerungen

Tipp: Ihr Logo und Ihre Markenfarbe finden Sie unter Einstellungen → "Logo und Farbe"; das Designstudio baut darauf auf.
