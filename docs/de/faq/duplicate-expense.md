So duplizierst du eine Ausgabe:
1. Öffne die Ausgabe, die du kopieren möchtest
2. Finde in der Seitenleiste die Karte „Aktionen"
3. Klicke auf „Duplizieren"
4. Eine neue Ausgabe wird mit denselben Daten erstellt
5. Passe Datum, Betrag oder andere Details an
