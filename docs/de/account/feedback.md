# Feedback

Teilen Sie Ihre Gedanken, melden Sie Fehler oder fordern Sie neue Funktionen direkt aus MyCompanyDesk heraus an.

## Feedback einreichen

1. Klicken Sie auf Ihr **Profilmenu** in der Seitenleiste (oder im mobilen Menu)
2. Wahlen Sie **Feedback**
3. Wahlen Sie eine **Kategorie**:
   - **Allgemein** — Allgemeine Kommentare oder Fragen
   - **Fehler** — Melden Sie etwas, das nicht korrekt funktioniert
   - **Funktion** — Schlagen Sie eine neue Funktion oder Verbesserung vor
   - **Bedienbarkeit** — Teilen Sie Feedback zur Benutzererfahrung
4. Bewerten Sie optional Ihre Erfahrung (1-5 Sterne)
5. Schreiben Sie Ihre **Nachricht** mit der Beschreibung des Feedbacks
6. Klicken Sie auf **Absenden**

Nach dem Einreichen sehen Sie einen Bestaetigungsbildschirm. Ihr Feedback wird an das MyCompanyDesk-Team zur Pruefung gesendet.

## Was als Naechstes passiert

Das Team prueft alle Feedback-Einsendungen:

- **Fehlermeldungen** werden priorisiert und untersucht
- **Funktionsanfragen** werden fuer die Produkt-Roadmap beruecksichtigt
- **Bedienbarkeits-Feedback** hilft bei der Verbesserung der Oberflaeche und Arbeitsablaeufe

## Tipps

- Seien Sie so spezifisch wie moeglich — beschreiben Sie, was Sie getan haben und was Sie erwartet haben
- Erwaehnen Sie bei Fehlern die Seite, auf der das Problem aufgetreten ist (dies wird automatisch erfasst)
- Sie koennen Feedback von jeder Seite in der App einreichen
