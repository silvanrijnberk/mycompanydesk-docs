# Profiel

Beheer je persoonlijke accountinstellingen, voorkeuren en meldingen.

## Persoonlijke informatie

Werk je profiel bij via de **Profiel**-pagina:

- **Naam** — Je weergavenaam
- **E-mail** — Je inlog-e-mail
- **Profielfoto** — Upload een foto of avatar
- **Taal**: De interface van MyCompanyDesk is Nederlandstalig. Er is geen interfacetaalinstelling. De documenttaal (voor facturen, offertes en e-mails) stel je per document in via Bedrijf → Facturatie.

### Taal

De interface van MyCompanyDesk is Nederlandstalig. Er is geen interfacetaalinstelling. Facturen, offertes en e-mails kunnen in het Nederlands, Engels, Duits of Frans worden opgesteld via Bedrijf → Facturatie (pdf-taal).

Je wachtwoord beheer je niet meer via Profiel maar via [Veilig inloggen](/nl/account/security).

## Meldingsvoorkeuren

Bepaal welke meldingen je ontvangt en hoe:

### E-mailmeldingen

- **Factuur betaald** — Wanneer een klant een factuur betaalt
- **Offerteaanvraag ontvangen** — Wanneer iemand een offerte aanvraagt via je bedrijfspagina
- **Teamactiviteit** — Wanneer teamleden acties uitvoeren
- **Uitgave goedgekeurd** — Wanneer een uitgave is goedgekeurd

### E-mailsamenvatting

Kies hoe vaak je samenvattings-e-mails ontvangt:

- **Dagelijks** — Dagelijkse samenvatting van activiteit
- **Wekelijks** — Wekelijks overzicht
- **Maandelijks** — Maandelijks overzicht
- **Uit** — Geen samenvattings-e-mails

### Pushmeldingen

Schakel browserpushmeldingen in voor realtime waarschuwingen wanneer je de app geopend hebt.

### In-app-meldingen

Realtime meldingen binnen de MyCompanyDesk-interface. Deze staan altijd aan en verschijnen bij het meldingsbel-icoon in de header.

## Gekoppelde apps

Bekijk en beheer OAuth-verbindingen:

- **Google** — Gekoppeld voor inloggen en/of Gmail-verzending
- **Microsoft** — Gekoppeld voor inloggen en/of Outlook-verzending

Trek de toegang in om een app los te koppelen van je account.

## Tijd en reizen

`/me/time-travel`

Persoonlijke standaarden voor de tijdregistratie- en reismodules. De pagina heeft twee tabs:

- **Tijd** (`/me/time-travel/time`) — Persoonlijke overschrijvingen voor tijdregistratiestandaarden.
- **Reizen** (`/me/time-travel/travel`) — Thuisadres, NS-Business OV-chip, standaardauto, kilometervergoeding.

### Tijdoverschrijvingen

Via de Tijd-tab kun je werkruimtebrede tijdregistratiestandaarden overschrijven voor je eigen registraties. Elk veld heeft een wisknop om terug te keren naar de werkruimtewaarde. Alle wijzigingen worden automatisch opgeslagen.

| Instelling | Wat het bepaalt |
|---|---|
| **Uurtarief** | Je standaardtarief voor nieuwe tijdregistraties |
| **Afrondingsinterval** | Rond de duur af op stappen van 15, 30 of 60 minuten |
| **Project verplicht** | Blokkeer opslaan zonder project |
| **Omschrijving verplicht** | Blokkeer opslaan zonder omschrijving |
| **Standaard regelomschrijving** | Vul het omschrijvingsveld voor bij nieuwe registraties |
| **Factuuromschrijvingsformaat** | Sjabloon voor opmaak van tijdregistratieregels op facturen |
| **Servicekosten** | Vaste kostenregel (omschrijving, bedrag, BTW-tarief) automatisch toegevoegd aan facturen met tijdregistraties |
| **Dag start / einde** | Grenzen van de werkdag op de tijdlijnweergave |
| **Pauze** | Pauzeweergave aan/uit plus starttijd en duur |

Wanneer een persoonlijke waarde is ingesteld, gaat deze voor op de werkruimtestandaard. Zie [Tijdregistratie > Persoonlijke overschrijvingen](/nl/features/time-registration#persoonlijke-overschrijvingen) voor de volledige cascade-regels.
