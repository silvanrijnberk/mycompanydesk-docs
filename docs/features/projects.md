# Projects

Organize your work into projects to track income, costs and results per engagement.

## Overview

Projects group invoices, expenses and time entries together. This gives you a clear picture of how each project or client engagement is doing financially.

::: info
The projects module is optional. Enable it in **Settings > Onderdelen** if you don't see it in the navigation.
:::

## Creating a project

1. Go to **Projects** and click the new project button
2. Fill in:
   - **Name**: A clear project name
   - **Customer**: The client this project is for
   - **Status**: Planned, Active, Paused, Completed or Cancelled
   - **Description**: What the project involves
   - **Budget**: An optional budget amount, plus an optional hourly rate
   - **Notes**: Anything you want to remember about the project
3. Click **Save**

You can also give a project its own number, start and end date, a colour, and a custom invoice line description format for hours billed from this project.

## Project statuses

| Status | Description |
|---|---|
| **Planned** | Agreed, but work has not started yet |
| **Active** | Currently being worked on |
| **Paused** | Temporarily on hold |
| **Completed** | Finished, no more work expected |
| **Cancelled** | Called off |

Archiving is separate from status: any project can be archived from the list or the detail page to keep your active list clean, and restored later.

## Project detail page

The project detail page shows:

- **Income, expenses and result**: Totals from the invoices and expenses linked to this project
- **Linked invoices**: All invoices related to this project, with a shortcut to create a new one
- **Linked expenses**: Costs attributed to this project
- **Notes**: Free-form notes on the project

## Linking items to projects

When creating invoices, expenses, or time entries, select the project from the dropdown to link them. This automatically updates the project's financials.

- **Invoices**: Track revenue per project
- **Expenses**: Track costs per project
- **Time entries**: Track hours per project

## Budget

The budget is a simple reference amount you set on the project. It appears in the project's quick view on the list, so you can hold your spending and invoicing against it at a glance.

## Bulk actions

Select multiple projects to **Archive** or **Delete** them at once. The Archive view offers restore or move to trash; the Trash view offers restore or permanent deletion.

## Tips

- Use projects even for small engagements: the insights are valuable
- Link both invoices and expenses to get an accurate result per project
- Archive completed projects to keep your active list clean
