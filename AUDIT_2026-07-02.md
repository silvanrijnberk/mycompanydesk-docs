# Docs Audit — 2026-07-02

Full accuracy + prose audit of the English source pages in `docs/` against
`mycompanydesk` (`origin/development` @ `a5d4239a7`, 2026-07-02), plus a
structural and human-voice pass over the nl/de/fr mirrors. Docs were last
updated 2026-06-04; **164 merges** landed on the app since.

> **Method:** six parallel audit passes — (1) gap audit over every merge since
> 2026-06-04, (2-5) page-by-page accuracy verification against app code
> (routes, i18n labels, components, API services) for invoicing/quotes,
> expenses/bank/reports/time, website/inbox/builder, and settings/account,
> (6) prose/human-voice + locale integrity. Every finding cites app-code
> evidence; "verified clean" lists are included so silence ≠ unchecked.

## Verdict

**The docs are substantially out of date, and the biggest single cause is
timing:** docs PR #144 (2026-06-04) repointed everything at the
`/workspace/*` + `/me/*` split IA — which the app deleted **one week later**
(2026-06-11, settings simplification PRs #4349/#4350/#4354/#4360). Several
pages now state the exact inverse of reality. On prose: the classic AI-smell
sweep came back clean, but the corpus is visibly two interleaved corpora
(human-ish FAQ/guides vs machine-synced engineering notes published verbatim),
several pages leak internal implementation details, and the locale mirrors
carry systematic machine damage (stripped diacritics, untranslated titles,
a 9-file mirror gap).

## 2. Actively harmful pages (fix first)

1. **`faq/two-factor-auth.md`** — says "Two-factor authentication is
   currently not available in this app build". TOTP + backup codes +
   passkeys + remember-this-device shipped (#4842, #4833); TOTP is a login
   second step with a `recover-2fa` flow. Worst factual error in the corpus.
2. **`faq/delete-account.md`** — sends users to a security page that has no
   danger zone; deletion lives at `/settings/opzeggen`. Users dead-end.
3. **`settings/index.md`** — "The old flat `/settings/*` URLs are gone" —
   exactly inverted; whole page describes the deleted IA.
4. **`advanced/keyboard-shortcuts.md`** — almost entirely fictional: no
   `G then D` nav, no `/`, `?`, `J/K/X`, `A/E/S` keys exist (the app even
   ships a typing-focus-warning plugin incompatible with letter shortcuts).
   Real shortcuts: Cmd/Ctrl+K palette, Esc back-nav, builder undo/redo.
5. **`advanced/api.md`** — wrong auth header (`Authorization: Bearer` →
   actual **`X-API-Key`**); `POST /invoices/:id/send` and `mark-paid` don't
   exist (send = `POST /invoices/:id/email`); webhook events list wrong;
   webhooks live at `/settings/webhooks`; keys have scopes/IP allowlist/expiry.
6. **`settings/email.md` + `faq/email-template.md`** — document per-type
   template editing that was removed; tile/state-machine IA gone; email
   sending settings are now **ungated** on `/settings/email` (only own-domain
   sending is Pro).
7. **`faq/multi-currency.md` + `faq/change-currency.md`** — point at a
   currency preference that doesn't exist; platform is fixed nl-NL/EUR.
8. **`settings/billing.md`** — seat add-ons retired (Pro = unlimited access);
   Starter now includes contracts/properties/projects (#4865); free-column
   errors (receipt scanning, assistant chat, AI insights are on Free);
   `/settings/billing` redirect claim reversed; **TODO comment shipped in
   the middle of the pricing table** (values now verifiable against
   `plans.config.js` — the AI-caps rows are correct as written).
9. **`getting-started/company-setup.md` + `features/setup-wizard.md`** —
   document the retired 5-step Business/Registry/Domain/Magic/Review wizard.
   Live wizard: **trade → goals/intent (#4854) → KVK → (domain → brand, only
   with the website goal) → Founding Members**. Country registries beyond
   KVK/NL are gone; the Review step is gone.
10. **`faq/change-language.md` + `account/profile.md` + intro** — "interface
    is Dutch-only" is now false: Settings → Weergave has an nl/en/de/fr
