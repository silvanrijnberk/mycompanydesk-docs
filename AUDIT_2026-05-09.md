The overall state is **majority stale** because two large refactors landed
since the last sync (`docs/sync-20260410`):

1. **Settings reshape.** `/settings/*` and `/company/*` were collapsed into
   a `/workspace/...` umbrella (Waves 0ŌĆō4) and personal account pages moved
   to `/me/...`. Almost every "Go to Settings > X" instruction in the docs is
   now wrong.
2. **VAT page rebuild.** The NL VAT page was rewritten as a 5-tab surface
   (Overzicht / Aangifte / Transacties / Planner / Regels) with hero deadline
   ring, KOR toggle, KIA tracker, ICP-opgaaf, OSS, foreign-VAT refund,
   suppletie + priv├®-onttrekking corrections, pre-filing checks and an
   AI assistant. Docs still describe a single-page period-lock view.
3. **Plan rename.** `Enterprise` is now `Business` end-to-end (commit
   `50493531a`). Docs already use "Business" but mention a legacy
   free-plan pool (`Starter / Enterprise`) that no longer matches the UI.
4. **AI dashboard summaries removed** (commit `184d656a9`). Docs still
   document them.
5. **Quotes split** out of invoices into their own table + module + routes
   (commits `aa037c3e1` ŌåÆ `f9085fc42`). Public-facing behaviour is unchanged
   so docs remain mostly fresh, but several internal cross-links assume the
   old conjoined model.
6. **Expenses gained** workspace-managed categories, multi-rate JSONB lines,
   per-expense `vat_treatment` override, depreciation engine, magic-setup
   wizard. Docs still describe a single-VAT-rate-per-expense form.
